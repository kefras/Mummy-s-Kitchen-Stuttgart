# 🍽 Mummy’s Kitchen Stuttgart | Luxury Catering Website

Welcome to the **Mummy’s Kitchen Stuttgart** luxury catering website project – a **premium dark luxury website** built to showcase authentic Nigerian cuisine, European dishes, and upscale catering services for weddings, corporate events, and family celebrations.

This project transforms a flyer-style branding concept into a fully interactive, modern, mobile-responsive website.

---

## 🌟 Features

* **Hero Section:** Elegant banner inspired by flyer design
* **3D Hover Cards:** Showcase catering services in style
* **Online Menu Ordering:** Users can select packages and submit booking requests
* **About Section:** Brand story and premium catering overview
* **Services Section:** Weddings, corporate events, and family gatherings
* **Pricing/Menu Packages:** Buffet and plated services with clear pricing
* **Testimonials:** Event client feedback for trust building
* **Luxury Gallery:** Highlight food presentation visually
* **FAQ Accordion:** Common questions for catering services
* **Contact & Booking Form:** Easy customer engagement
* **Dark Luxury Design:** Modern, visually appealing, food-first aesthetic
* **Fully Responsive:** Optimized for desktop and mobile

---

## 🖥️ Tech Stack

* **HTML5 & CSS3:** Markup and styling
* **JavaScript:** Interactivity for order forms and FAQ toggles
* **Responsive Design:** Mobile-first with flexible grids
* **3D Card Effects:** CSS transform for hover animation

## 🚀 How to Run Locally

1. **Clone the repository**

```bash
git clone https://github.com/<your-username>/mummys-kitchen-stuttgart.git
```

2. **Navigate into the folder**

```bash
cd mummys-kitchen-stuttgart
```

3. **Open the website in your browser**

```bash
open index.html   # macOS
# or
start index.html  # Windows
```
